1. 链表

前置知识：指针。
我们来瞅一眼OI Wiki上对链表的讲解：

链表和数组都可用于存储数据，其中链表通过指针来连接元素，而数组则是把所有元素按次序依次存储。
同时，它也给出了两种数据结构的利与弊：
链表可以方便地删除、插入数据，操作次数是O(1)O(1)O(1)。但也因为这样寻找读取数据的效率不如数组高，在随机访问数据中的操作次数是O(n)O(n)O(n)。
数组可以方便的寻找读取数据，在随机访问中操作次数是O(1)O(1)O(1)。但删除、插入的操作次数却是却是O(n)O(n)O(n)次。

单向链表

我们来看看单向链表：

typedef struct
{
    int val;
    node *next;
} node ;

单向链表中包含数据域和指针域，其中数据域用于存放数据，指针域用来连接当前结点和下一节点。

void insertNode(int i, Node *p)
{
    Node *node = new Node;
    node->value = i;
    node->next = p->next;
    p->next = node;
}

上面这串代码是在p和p->next之间插入一个值为iii数据。

理解一下，我们就开始讲下一个内容了。
